enum xyplot::eFLASHSPEED

Скорость мерцания профиля или группы профилейв режиме выделения SM_FLASH.

Элементы перечислений
FLS_NO_FLASH 

Нет мерцания

FLS_VERY_SLOW 

Очень медленно (1200 ms - отображение профиля, 600 ms - профиль не отображается)

FLS_SLOW 

Медленно (600 ms - отображение профиля, 400 ms - профиль не отображается)

FLS_NORMAL 

Нормально (400 ms - отображение профиля, 300 ms - профиль не отображается)

FLS_FAST 

Быстро (300 ms - отображение профиля, 150 ms - профиль не отображается)

FLS_VERY_FAST 

Очень быстро (150 ms - отображение профиля, 75 ms - профиль не отображается)

FLS_EXTREME 

Экстремально (10 ms - отображение профиля, 10 ms - профиль не отображается)
