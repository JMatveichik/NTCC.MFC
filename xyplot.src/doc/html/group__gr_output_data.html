Дописать данные для профиля

Функция дописывает данные содержащиеся в массивах px и py для профиля.

Аргументы
[in]hPlotдескриптор экземпляра 2D графика содержащий профиль
[in]hProfileдескриптор профиля
pxуказатель на массив данных, содержащий координаты профиля по оси X
pyуказатель на массив данных, содержащий координаты профиля по оси Y
sizeколичество точек дописываемых из массивов
Возвращает
true если удалось дописать данные для профиля
false если не удалось дописать данные для профиля
Прим.
Размеры массивов px и py должны быть одинаковыми.

Копирует данные профиля в массивы

Аргументы
[in]hPlotдескриптор экземпляра 2D графика содержащий группу профилей
[in]hProfileдескриптор профиля из которого копируются данные
[out]pxуказатель на массив данных, содержащий координаты профиля по оси X
[out]pyуказатель на массив данных, содержащий координаты профиля по оси Y
[out]sizeразмер масивов данных px и py
Возвращает
true если удалось получить данные для профиля
false если не удалось получить данные для профиля
Прим.
Для определения необходимых размеров массивов необходимо вызвать функцию с параметрами px = NULL и py = NULL.
