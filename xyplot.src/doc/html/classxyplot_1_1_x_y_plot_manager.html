Класс-обертка для работы с 2D графикой

Класс-обертка для работы с объектами отображения 2D графики. Данный класс предоставляет инструменты создания, манипуляции и уничтожение объектов 2D графики. Класс является сингелтоном , т.е. существует только один экземпляр данного класса, доступ к которому осуществляется через статическую функцию Instance.
