Создания экземпляра региона

Создания прямоугольной области графика ограниченного линиями с координатами [xFrom, yFrom] и [xTo, yTo].

Аргументы
[in]hPlotдескриптор экземпляра 2D графика для которого создается регион
axisXось X, для которой создается регион
axisYось Y, для которой создается регион
xFromначальное значение региона физических единицах по оси X
xToконечное значение региона физических единицах по оси X
yFromначальное значение региона физических единицах по оси Y
yToконечное значение региона физических единицах по оси Y
pszTitleподпись региона

дескриптор экземпляра региона

Задание новых границ региона

Границы региона прямоугольной области графика перемещаются в новое положение xFrom, yFrom, xTo, yTo.

Аргументы
[in]hPlotдескриптор экземпляра 2D графика для региона
hRegionдескриптор региона
xFromновое начальное значение региона в физических единицах по оси X
xToновое конечное значение региона в физических единицах по оси X
yFromновое начальное значение региона в физических единицах по оси Y
yToновое конечное значение региона в физических единицах по оси Y
Возвращает
true если удалось поменять значение границ региона
false если не удалось поменять значение границ региона
